## Overview
**CopperStripper** is a lightweight Fabric mod that enhances copper maintenance in Minecraft. Hold an axe or brush in one hand and a copper block in the other, then right-click to instantly remove oxidation or wax.

## Commands
All commands require OP level 2 permission.
| Command                           | Description          |
| --------------------------------- | -------------------- |
| /easeon copperstripper            | View current status  |
| /easeon copperstripper <on/off>   | Enable/disable       |

<br/>

## Configuration
```json
{
  "enabled": true,
  "requiredOpLevel": 2 // Requires a server restart to take effect.
}
```
`config/easeon/easeon.ss.copperstripper.json`
